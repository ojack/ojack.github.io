Flight of the Condors

30. April 2013

Animated map of flight paths of the California Condor, with a side-panel that shows the “condor-eye-view” as rendered in Google Earth.

Process:

The animated paths were created in Processing, by plotting a the location at each incremental hour of the Condor’s flight, and then saving the frame. The thickness of the path at each point varies according to the elevation of the underlying terrain, giving the paths a hand-drawn, organic feel. I used GoogleEarth Javascript API to capture the view at each coordinate and angle of view (tangent) along the path. A NodeJS server communicated between the Processing App and the Javascript API to keep them in sync, and communicated with terminal to take a screenshot of the Google Earth window at each point in the Condor’s trajectory. Finally, I assembled the 100,000 frames into a video.
